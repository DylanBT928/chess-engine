# Chess Engine ♜

A fully functional chess game built using Python and Pygame. This project features an interactive GUI, legal move validation, and a smooth user experience for playing chess.


## Dependencies 🛠️

- Python 3.x
- Pygame
- Python-Chess

Install them using:

```
pip3 install pygame
pip3 install chess
```
